# Daily Routine Tracker

A simple click-through routine tracker with implementation intentions.

## Features

- **Main Tracker Page**: Red screen with white text showing current task. Click to turn green and advance to next task.
- **Notes Page**: All rules, backup plans, affirmations, and schedule details.
- **Implementation Intentions**: Each task shows the if-then plan below it.

## Setup for GitHub Pages

2. Upload these files to the repository:

   - `notes.html`

4. Under "Source", select "main" branch

## Usage

- Click anywhere on the red screen to mark a task as complete
- Screen turns green for 1 second, then shows next task
- Click "Notes & Rules" in top right to view all details
- Refresh page to restart from beginning
