Daily Routine Notes & Rules

Core Rules

No caffeine after 3pm
If work isn't finished after work block 2, then it waits until tomorrow (or later if I choose)
If still working at 8pm, then stop work for 5 min to eat, then continue if needed
If muay thai runs late, everything after shifts back (only the 9:45am meeting is static)

Starting Ritual (Before Each Work Block)

Every time before a work block, do this exact sequence:

  1. Close all tabs except work
  2. Put phone on silent
  3. Message girlfriend: "Working now"
  4. Set timer for 30 minutes
  5. Tell yourself: "Just 2 minutes"
  6. Start

Backup Plan

If I wake up late:

  1. Shower
  2. Coffee
  3. Be at meeting at 9:45
  4. After meeting: Do missed morning steps (affirmations, make bed, breakfast, plan day)

Schedule Overview

Morning Routine

  • Wake time: 7:15am
  • Total time: 1 hour 45 minutes (15 min to wake up + 1 hour 30 min routine)

Work Day Key Times

  • 9:00am - Prep for meeting
  • 9:45am - Meeting (30 min) - STATIC, NEVER MOVES
  • 8:00pm - Last meal (nothing after this)

Evening Routine

  • Bedtime window: 9-11pm
  • 1 hour before bed: Shower, teeth, skincare
  • 30 min before sleep: Reading OR YouTube "to watch list"

Flexible Elements

  • Short productive break: Beach, shops, tidying, etc.
  • Free time (1 hour): Truly free - enjoy yourself
  • Muay Thai: Optional, based on energy and digestion
  • Bedtime: Flexible between 9-11pm
